Visualize dimension reduction regression using a coplot

Description

This function produces a coplot of the response versus specified directions in dimension reduction regression.

Usage

dr.coplot(object, which=1:object$numdir, mark.by.y=F, ...)

Arguments

object a dr object
which specify the indices of the directions to be plotted
mark.by.y if FALSE, plot the response versus the first specified direction conditioning on the second specified direction. If TRUE, plot first direction versus second direction conditioning on the third direction, using the response as a marking variable to color the points.
... other arguments passed to coplot

Details

The function dr.permutation.test.statistic is not to be called by the user, but it computes the test statistic for each different type of dr object.

Value

A coplot.
