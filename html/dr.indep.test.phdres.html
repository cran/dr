Test of independence for pHd based on residuals

Description

Compute the p-value for the hypothesis that the central subspace for OLS residuals has dimension 0 against the alternative that it is greater than zero. This is an internal method not usually needed by the user.

Usage

dr.indep.test.phdres(object, stat)

Arguments

object phdres object
stat value of the test statistic

Author(s)

Sanford Weisberg <sandy@stat.umn.edu>
